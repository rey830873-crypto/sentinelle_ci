# 🛡️ SentinelleCI - Votre voix, votre commune

SentinelleCI est une plateforme citoyenne innovante conçue pour réduire la fracture entre les citoyens ivoiriens et leurs autorités locales.

## 🌟 Pourquoi nous allons gagner ? (Arguments Clés)

### 1. 🔗 Transparence Blockchain (Polygon)
Contrairement aux applications classiques, chaque signalement sur SentinelleCI génère une preuve immuable sur la blockchain. **Impact :** Suppression impossible des données, responsabilité totale de la commune.

### 2. 🤖 IA & Accessibilité
- **Signalement Vocal :** Transcription par IA pour les citoyens préférant l'oralité.
- **Détection IA Vision :** Catégorisation automatique des problèmes via analyse d'image.
- **Inclusivité :** Interface disponible en Français, Dioula et Baoulé.

### 3. 🎮 Gamification & Engagement
Un système de points et de badges (Leaderboard) encourage les citoyens à devenir des "Vigies de quartier" actifs, transformant le civisme en une expérience engageante.

### 4. 📊 Dashboard Décisionnel pour la Mairie
Une interface d'administration permettant de visualiser les "Heatmaps" (zones d'urgence) et de gérer les interventions en temps réel.

## 🏗️ Architecture Technique
- **Framework :** Flutter (Multi-plateforme)
- **Pattern :** MVVM (Model-View-ViewModel) pour une scalabilité maximale.
- **State Management :** Provider.
- **Backend Ready :** Architecture prête pour Firebase / Supabase.
